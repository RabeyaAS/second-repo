My First Website Using Git in VS with Powershell

NOTE: You only add the whole repository when creating a New Repository for the first time. This goes for everytime you create a new file and you can do that either in CMD (Command Prompt) or in VS Terminal -> Powershell or CMD and once done you don't need to repeat it again.

How To Push Incremental Code

Once done creating a new repository you can now type in codes and for every little change you make you only add 3 things to push the changes into github:

  1. git add .
  2. git commit -m" "
  3. git push
